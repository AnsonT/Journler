Use your keyboard to accomplish tasks quickly in Journler. To find the shortcuts for common commands look in the menus. Many items in Journler also have contextual menus, including entries, folders, resources, entry content and the calendar. To see a contextual menu, press the Control key and click once on the item or right-click once.

A number of keyboard shorcuts in the main journaling window are hidden from view, especially those that make it easy to navigate your journal. Use the key combinations below to take advantage of them.

Looking for information about modifiers? Read about Journler's keyboard modifiers.

Navigate the Calendar

To Today's Date Command-Return
Snap to Date of Selected Entry Command-Control-Return

Previous Day Command-Option-Left Arrow
Next Day Command-Option-Right Arrow
Previous Day with Entries Command-Control-Left Arrow
Next Day with Entries Command-Control-Right Arrow
Previous Month Command-Option-Control-Left Arrow
Next Month Command-Option-Control-Right Arrow

Navigate Your Folders

Select Journal Command-Option-Return
Previous Folder Command-Option-Up Arrow
Next Folder Command-Option-Down Arrow

Navigate the Entry Browser
(Leopard only)

Previous Entry in Browser Command-Control-Shift-Up Arrow
Next Entry in Browser Command-Control-Shift-Down Arrow

Change the Keyboard Focus

Focus to Folders List Command-6
Focus to Entries List Command-7
Focus to Entry Text Command-8
Focus to Resource List Command-9

Other Shortcuts

Delete the Selection Command-Delete
Empty the Trash Command-Shift-Delete
