Speakable Items

Table of Contents

Previous | Next

If you have a microphone attached to your computer, you may speak commands to Journler. Powerbooks and iBooks include built-in microphones. Other Macs may also have built-in mics. If you are uncertain, open System Preferences and under Sound check Input.

Speakable items is disabled by default. To active Journler's speakable items and enable voice commands, select Toggle Speakable Items from the File menu. The floating speakable window will appear on the screen. Normally you hold the escape key and begin speaking. Journler supports the following commands. Speak the words clearly into your mic.

Creating New Objects

Create New Entry
Opens the new entry dialog requesting a title, tags and category.

Create New Tab
Creates a new tab and selects it.

Create New Folder
Opens the new folder dialog requesting the title of a new regular folder.

Create New Smart Folder
Opens the new smart folder dialog requesting the criterion for a new smart folder.

Working with the Selection

New Tab with this Entry
Creates a new tab with the currently selected entry and goes to it.

New Window with this Entry
Creates a new window with the currently selected entry and brings it to the fore.

Send Entry to iWeb
Sends the currently selected entry to iWeb for blogging.

Blog this Entry
Opens the blog center with the current entry.

Email Selection
Opens your default mail client with the currently selected entry.

Print Selection
Prepares the current entry for printing.

Export Selection
Shows the export entry save dialogue box.

Adding Media to an Entry

Record Audio
Opens the audio recorder.

Record Video
Opens the video recorder.

Take a Picture
Opens the snapshot window.

Show Media
Opens iLife media integration.

Show Contacts
Opens Address Book integration.

Bulk Commands

Import Files
Begins the document import process.

Print Journal
Begins the process that prints a selected range of entries from the journal.

Export Journal
Opens the export journal sheet, requiring a range of entries to export and their format.

Navigating the Calendar

Go to Today
Navigates the calendar to today and displays the day's first entry.

Go to Yesterday
Navigates the calendar to the previous day.

Go to Tomorrow
Navates the calendar to the next day.

Go to Previous Month
Navigates the calendar to the previous month.

Go to Next Month
Navigates the calendar to the following month.

File Menu Commands

Save
Saves the current selection.

Close Window
Closes the foremost window.
