In addition to keyboard shortcuts, Journler relies on modifier keys to implement a number of power user features. To take advantage of a modifier key combination, hold down the appropriate key or keys while simultaneously using the mouse or giving your command. Continue holding the keys until the command has been executed.

For example, to select mutiple entries in the browse table hold down the shift or command key while clicking with the mouse on the desired entries.

Looking for information about hidden shortcuts? Read about Journler's keyboard shortcuts.

The Browse Table

Select multiple continuous entries Hold Shift
Select multiple discontinuous entries Hold Command

Open an entry in a new tab Hold Command-Shift
Open an entry in new window Hold Command-Option

The Resource Pane

Select multiple continuous resources Hold Shift
Select multiple discontinuous resources Hold Command

Open a resource in a new tab Hold Command-Shift
Open a resource in new window Hold Command-Option
Open a resource with the Finder Hold Option

The Source List (Your Folders)

Keep filter or search active Hold Option

Dragging media to an entry

Force the item to link, disregarding preferences Hold Control
Force the item to copy, disregarding preferences Hold Option
(Please note that it is not always possible to override the default setting, i.e. Mail messages must be copied.)

Acting on the selection
The following shortcuts work on the selected entry or resource when the table has keyboard focus.

Open in a new window Press Return
Open in a new tab Hold Shift and press Return
Open in a new floating window Hold Option and press Return
