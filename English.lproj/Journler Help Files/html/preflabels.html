Label Names

Personalize the names of the various color labels supported by Journler. Journler uses these names in the label menus which appear across the program.

Draw labels in bookmarks bar

Enable this option and Journler also draw an entry's label in the bookmarks bar. The bookmarks bar appears at the top of Journler's main window just underneath the toolbar. If you do not see it, choose Show Bookmarks Bar from the View menu.
