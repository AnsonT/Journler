Journal Name

Change the name of your journal.

Default Category

New entries use this category if you do not otherwise give them one. Set the default category to a dash "-" for a blank default category.

Drop Box Category

Items coming in through the services menu, drop box or dock drop use this category. Set to a dash "-" for a blank default category.

Start Week On

The first day of the week in the calendar.

Autosave Timelapse

Specify a time in minutes. Journler autosaves the current selection at this interval. Journler also saves your data anytime you change the selection.

Maximum Width of Embedded Images

The maxim horizontal length for the image representation embedded in your entry. The actual image is not affected. Check Use full size to use the image's actual size.

New Entry Date

When backdating entries be sure to check The date currently selected in the calendar. Otherwise Journler switches to today's date before creating the new entry.
